## Dependências do projeto:

* node.js 12 ou superior
* docker

## Instalação

```bash
git clone https://github.com/codenation-dev/squad-1-ad-nodejs-kroton-1
cd squad-1-ad-nodejs-kroton-1
npm install
docker-compose up -d
npm sync
npm start
```
